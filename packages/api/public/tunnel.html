🔗 ProxVN Tunnel

Expose local services to the internet — HTTP/TCP/UDP Tunneling Platform

🌐

HTTP Tunnel

Auto SSL subdomain, HTTPS proxy. Expose web apps với domain riêng.

🔌

TCP Tunnel

SSH, RDP, Database forwarding. Kết nối mọi service qua TCP.

📡

UDP Tunnel

Game servers, real-time apps. Low-latency UDP forwarding.

TLS 1.3

End-to-end encryption. Certificate pinning, JWT authentication.

📁

File Sharing

WebDAV-based secure file sharing với auth và permissions.

📊

Dashboard

Web dashboard quản lý clients, bandwidth, kết nối real-time.

🖥️ Server

# Khởi động server
proxvn-server.exe -port 8881

# Dashboard: http://localhost:8881/dashboard/
# Tunnel port: 8882 (TLS)
# Default: admin / admin123

Server quản lý connections và routing cho clients.

💻 Client

# Kết nối và expose port 3000
proxvn-client.exe -server localhost:8882 -local 3000

# Expose SSH (port 22)
proxvn-client.exe -server localhost:8882 -local 22 -type tcp

Client chạy trên máy cần expose services.
